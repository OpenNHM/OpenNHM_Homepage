As already announced, here’s some more info about the

2nd public AvaFrame workshop on 18th of January at 9:00 (Central European Time; UTC+1:00).

It takes place via an MS Teams meeting: TEAMS-LINK (Login via browser is possible, no app installation necessary, no account required). The workshop will be held in English.

If you want to follow the hands on demo:

  • please prepare by following the installation instructions on our documentation: installation. We will only concentrate on the QGis / GUI mode of handling AvaFrame.

  • you can bring your own data if you want, but we will also provide test data.

Update Download test data (zip)

For anybody unable to participate we will provide a recording of the workshop, and you can reach us on the AvaFrame matrix channel. For more information see here: AvaFrame communications.
